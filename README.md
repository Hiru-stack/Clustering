### Customer Segmentation using K-Means Clustering
This project demonstrates how to perform unsupervised machine learning to segment customers based on their annual income and spending behavior. It utilizes the K-Means clustering algorithm, the Elbow Method for optimization, and PCA for data visualization.

🚀 Project Overview
The goal of this project is to categorize customers into distinct groups to help businesses tailor their marketing strategies effectively.

Algorithm: K-Means Clustering

Optimization: Elbow Method, Silhouette Score

Dimensionality Reduction: PCA (Principal Component Analysis)

Language: Python (scikit-learn, pandas, matplotlib)

🛠️ Tech Stack
Environment: Google Colab

### Libraries:

pandas (Data Manipulation)

scikit-learn (Clustering & Preprocessing)

matplotlib (Visualization)

📊 Workflow
Data Preparation: Generating/Loading customer data and applying StandardScaler.

Hyperparameter Tuning: Using the Elbow Method to determine the optimal number of clusters (k).

Model Training: Fitting the K-Means model to the data.

Visualization: Plotting the clustered data to identify trends and patterns.
